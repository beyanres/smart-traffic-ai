## 📌 عن المشروع

محاكاة تقاطع طرق ذكي يوضح كيفية منع **Gridlock (الاختناق الكلي)** باستخدام نظام إشارات مرورية متزامن.

| المرور | نظم التشغيل |
|--------|-------------|
| 🚦 Traffic Light | 🔒 Mutex / Semaphore |
| 🚗 Car | ⚙️ Process / Thread |
| 🛣️ Road | 💾 Shared Resource |
| 💥 Gridlock | 🛑 Deadlock |
| ✅ Prevention | 🛡️ Resource Ordering |

---

## 🎯 كيف يعمل

### ✅ مع Prevention
- **Phase 1:** North & South 🟢 | East & West 🔴 → سيارتان تعبران بأمان
- **Phase 2:** East & West 🟢 | North & South 🔴 → سيارتان تعبران بأمان
- **النتيجة:** 4 مركبات عبرت بنجاح، لا Deadlock!

### ❌ بدون Prevention
- جميع الإشارات 🟢 في نفس الوقت
- **النتيجة:** Gridlock! تصادم! 💥

## 🧠 المفاهيم المستخدمة

- **Deadlock Prevention** — منع حدوث الجمود قبل وقوعه
- **Circular Wait Prevention** — كسر حلقة الانتظار الدائرية
- **Mutual Exclusion** — حصرية استخدام المورد
- **شروط Coffman** — الشروط الأربعة للـ Deadlock
